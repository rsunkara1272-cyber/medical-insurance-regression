# medical-insurance-regression
Predicting medical insurance costs using Gradient Boosting with 95% R² and real-world insights
Medical Insurance Cost Prediction (Regression)
This project uses machine learning to predict medical insurance charges and uncover the key factors driving cost.
🔍 Key Insight
Smoking is the most powerful predictor of insurance cost.
Smokers pay ~2.2x more than non-smokers
The model identified this as the dominant feature without manual bias
⚙️ Models Used
Linear Regression (baseline)
Gradient Boosting Regressor (final model)
📊 Performance
R²: ~0.95
MAE: ~$1,400
Strong alignment between predicted and actual values
📈 Visualizations
The project generates:
Predicted vs Actual plot
Residual analysis
Feature importance chart
Smoker vs Non-smoker cost distribution
🧠 What this shows
This is not just a prediction problem—it demonstrates how machine learning can uncover real-world drivers of cost and risk.
